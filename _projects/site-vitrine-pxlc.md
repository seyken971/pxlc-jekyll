Le site vitrine léger permet de présenter une activité, des services, des coordonnées et des mentions légales sans dépendre d'un outil lourd.

Les contenus restent éditables en Markdown, le code est versionné dans GitHub et la publication peut être automatisée avec GitHub Pages.
